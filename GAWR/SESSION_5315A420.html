INCIDENT REVIEW

At 1638 hours, March 4, 2014, Englightened agent gongxiaoyin, rumored spouse of Resistance agent TywinLannister, was removed from Resistance Wuhan QQ group for being a non-Resistance agent. The removal was executed by agent SierraSix, present administrator of the Resistance Wuhan QQ group.

After the removal, several resistance agents, including agent amalucky, agent TywinLannister, and agent vvlong stated that it is not necessary to remove Enlightened agent gongxiaoyin from the group, because it has already become a group for general discussion. Agent Hexchain and agent SierraSix supported the removal, emphasising that the QQ group is intended for Resistance agents and Resistance agents only.

As dicussion seemed to reach an impasse, agent SierraSix initiated a vote to resolve the situation.
